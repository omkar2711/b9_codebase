Patient:
- GET /patients - Get all patients -> admin only
- GET /patients/:id - Get a specific patient by ID
- POST /patients - Create a new patient
- PUT /patients/:id - Update a patient's information
- DELETE /patients/:id - Delete a patient -> admin only
Doctor:
- GET /doctors - Get all doctors -> admin only
- GET /doctors/:id - Get a specific doctor by ID
- POST /doctors - Create a new doctor
- PUT /doctors/:id - Update a doctor's information
- DELETE /doctors/:id - Delete a doctor -> admin only
Appointment:
- GET /appointments - Get all appointments -> admin only
- GET /appointments/:id - Get a specific appointment by ID
- POST /appointments - Create a new appointment
- PUT /appointments/:id - Update an appointment's information
- DELETE /appointments/:id - Delete an appointment -> admin only    

Authentication:
- POST /auth/register - Register a new user (patient or doctor)
- POST /auth/login - Login a user and return a JWT token
- POST /auth/logout - Logout a user (invalidate JWT token)
